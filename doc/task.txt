Задача

Необходимо реализовать простое приложение с использованием фреймворка ​Angular2​ (можно смело взять 4 версию): мессенджер.
На главной странице приложения показан список пользователей (Имя, Фамилия),
приклике на кого-то из них список сдвигается влево разделяя экран надвое:
мы видимдетальную информацию о пользователе (О себе, возраст, город) и кнопку “Отправитьсообщение”.
По клику на отправку сообщения список пользователей долженсвернуться до максимально узкого состояния
оставив только иконки с инициалами,остальное место должно занять окно чата с формой отправки сообщения и
историейсообщений.

При отправке сообщение должно “упасть” в историю.
При обновлении страницы мы не должны терять состояние приложения(сохранять отправленное сообщение не нужно).
В качестве источника данных можно использовать JSON-файлы в статикеприложения.
Список пользователей и детальная информация -- разные типы данных.
Пользователей лучше сделать сразу много: 10-20 в самый раз.
В качестве верстки можно смело брать Twitter Bootstrap.

Оценка выполненияВ первую очередь будет оцениваться качество организации кода,
умение работать соструктурами данных,
эстетически-прекрасная организация внешнего вида и анимации.

Не нужно бояться привнести свою инициативу в интерфейс.
